# 함수에 대해서(Python)

### 1.함수란?
* 함수는 명령어들의 모음에 이름을 붙인 것이다.
* 함수는 작업에 필요한 데이터를 전달 받을 수 있으며, 작업이 완료된 후에는 작업의 결과를 호출차제에개 반환 할 수 있다.
* 함수가 필요한 이유는 이미 작성한 코드에서 몇 가지 내용이 차이가 나지 않는 새로운 코드를 만들려고 할때 기존 코드를 재활용며해서 간편하고 빠르게 바꿀 수 있기 때문이다.
***
### 2.함수를 쓰는 법
* 함수는
<pre><code>
  def 함수이름(매개변수1, 매개변수2,...):
    명령문1
    명령문2
</code></pre>
이렇게 쓸 수 있다.   
* 함수 호출이라는 것도 있는데 함수 호출이란 함수이름() 이렇게 쓴다면 함수가 호출되어 함수안에 있는 문장들이 실행되며 실행이 끝나면 호출한 위치로 되돌아간다.
함수와 함수 호출을 쓰는 예)
<pre><code>
  def get_area(radius):
    area=3.14*radius**2 #원의 면적
    return area #area로
  result=get_area(3) #함수안 매개변수(radius)한테 3을 넣음
  print(result) #결과는 28.26이 출력된다.
</code></pre>
