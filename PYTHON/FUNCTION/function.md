# 함수에 대해서(Python)

### 1.함수란?
* 함수는 명령어들의 모음에 이름을 붙인 것이다.
* 함수는 작업에 필요한 데이터를 전달 받을 수 있으며, 작업이 완료된 후에는 작업의 결과를 호출차제에개 반환 할 수 있다.
* 함수가 필요한 이유는 이미 작성한 코드에서 몇 가지 내용이 차이가 나지 않는 새로운 코드를 만들려고 할때 기존 코드를 재활용며해서 간편하고 빠르게 바꿀 수 있기 때문이다.
